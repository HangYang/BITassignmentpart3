Scope creep:
For the reason that the planning and designing stage of our project have lasted for a long time, most of the design and MVFs remained the same.
However, there are still some slight changes in certain areas. The game we planned at an early stage is Tetris, however, due to the Cascading Style Sheets conflict between running the tetris and the webpage itself, we changed the game from Tetris to Greedy Snake, which works fine with the website we developed.
Another scope creep is also in the gaming page, which we originally plan to create a database like MySql to store the ranking list, but due to the lack of experience, time and resources, we change it to a json database that only stores a small amount of data, but it also ensures the smooth operation of the system.
