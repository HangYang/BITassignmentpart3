1. The Database Administrator (DBA)

The database administrator (DBA) is a general term for the staff engaged in the management and maintenance of database management systems (DBMS). It belongs to a branch of operation and maintenance engineers. It is mainly responsible for the life cycle management of business databases from design, testing to deployment and delivery. The core goal of DBA is to ensure the stability, security, integrity and high performance of the database management system. This position is very important for a company and users. The user's data are stored in the database. Good management and protection of the database have a direct impact on the reputation or expenditure of a company. Data leakage will lose traffic, and improper management will cause more investment. This position requires the staff to have:

Skills and Experience:

  • basic theoretical knowledge, including hardware, operating system, network, storage, etc.

  • database basic knowledge, including interactive language (such as SQL), mode, paradigm, transaction, lock, etc.

  • database operation and maintenance skills, including deployment, backup, high availability, monitoring, migration, troubleshooting, patrol, etc.

  • database performance tuning skills, which is very important Including slow query analysis, index optimization, parameter tuning and so on.
2. Front-end Engineers

Front end engineers can be said to be the foundation of a website. In other words, this position is to write the code of the whole website, including various pictures, videos, animations, special effects and so on. This position requires the staff to have a lot of professional-level knowledge because this is the foundation of the website, you must have enough ability to ensure that the website runs 100% effectively.

Skills and Experience:

  • Front end engineers need to have expert knowledge of HTML / CSS / JavaScript

  • familiar with angular js/ Ember.js /Experience in jQuery or other class libraries

  • familiar with third-party component (plug-in) ecological environment and specific cases

  • familiar with jade/swing/handlebars/mustache or other template engines

  • familiar with sass Or other CSS preprocessor experience

  • experience in using CoffeeScript

  • good understanding and application of CSS / JavaScript Design Pattern

  • familiar with common data structures and algorithms

  • experience in using gruntjs / gulp js task runner

  • experience in using yeoman generator

  • bower / Volo / JSPM Experience in front-end static resource package manager

  • familiar with the local and remote debugging operation

  • experience in GIT.
3. Web Designer

A web designer is like a facade. From personal home pages to large companies, government departments and international organizations, all of them use web pages as their front page. When you click to the website, the first thing that comes into view is the interface design of the webpage, such as the introduction of the content, the placement of buttons, the combination of words, the application of colour, the guidance of use, etc. All of these are the domain of web design and the work of web designers. Web designers not only need to understand the design of the page,This position is very important. How can customers go in and have a look without a good facade?
